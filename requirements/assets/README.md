# Assets de requerimientos

Colocar aqui imagenes, capturas, bocetos, PDFs o referencias visuales del proyecto.

Ejemplos:

```text
requirements/assets/pantalla_clientes.png
requirements/assets/boceto_dashboard.jpg
requirements/assets/flujo_operativo.pdf
```

El Skill Frontend debe usar estos archivos como soporte visual. El archivo principal sigue siendo `requirements/project_requirements.yaml`.
